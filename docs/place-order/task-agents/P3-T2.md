Claim API
Endpoint: POST /internal/v1/coupon-claims
Checklist:
Input validation:

 idempotencyKey, couponCode, userId không rỗng
 orderAmount > 0

Idempotency:

 Query CouponClaims WHERE OrderIdempotencyKey = ?
 Tìm thấy → trả 201 với claimId cũ

Coupon validation:

 Coupon tồn tại → nếu không: 422 COUPON_NOT_FOUND
 IsActive = true → nếu không: 422 COUPON_INACTIVE
 ValidFrom <= now <= ExpiresAt → nếu không: 422 COUPON_EXPIRED
 orderAmount >= MinOrderValue → nếu không: 422 ORDER_BELOW_MIN_VALUE

Atomic claim (Redis):

 SETNX coupon:{code}:user:{userId} TTL 15min

 Trả 0 (key đã tồn tại) → 409 COUPON_ALREADY_USED


 DECR coupon:{code}:quota (nếu TotalQuota != null)

 Kết quả < 0 → INCR lại, DEL user key → 409 COUPON_EXHAUSTED



Persist:

 Tính DiscountAmount theo DiscountType
 INSERT CouponClaim: Status=CLAIMED, ExpiresAt = now + 15min

Response:

 201: { claimId, discountAmount, expiresAt }
